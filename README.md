## Project overview
This project implements the following sorting algorithms:
- Bubble sort
- Insertion sort
- Selection sort
- Quick sort
- Shell sort - Knuth Sequence
- Cocktail shaker sort
- Counting sort
- Merge sort
- Heap sort
- Radix sort
- Bitonic sort
- Quick Sort - Hoare Partition scheme
- Dealer
